To find the sum of all solutions of the equation p(x) : ax^2 +bx + c = 0 given the conditions on a, b, and c, we need to use the relationships derived from the problem statement.

Given:

  • b is a solution of ax^2 + c = 0

  • a is a solution of bx^2 + c = 0

  • c is a solution of ax^2 + bx = 0

Step-by-step Solution

  1. For b being a solution of ax^2 + c = 0:
    ab^2+ c = 0 ⟹ c = −ab^2
  2. For a being a solution of bx^2 + c = 0:
    ba^2 + c = 0 ⟹ c = -ba^2
  3. For c being a solution of ax^2 + bx = 0:
    c(ac +b) = 0 ⟹ ac + b = 0 ⟹ b = -ac

From the above equations, we have:
c = -ab^2 and c = -ba^2

Equating the two expressions for c:
-ab^2 = -ba^2 ⟹ ab^2 = ba^2 ⟹ a(b^2 - a) = 0

Since a =/ 0, we get:
b^2 = a

Using b = -ac, we substitue b = ± a^1/2 into the equation b = -ac:
a^1/2 = -a(-a(a^1/2)) = a

Hence, there are two possibilities for b:
b = a^1/2 or b = -a^1/2

Subtitute b = a^1/2:
a = b^2/a = b

Thus b = a or b = -a:

  • If b = a, then c = -ab^2 = -a(a^2) = -a^3
  • If b = -a, then c = -a(-a)^2 = -a^3

The quadratic equation is:
ax^2 + bx + c = 0

Given a and b =±a, the root sum S of the quadratic equation ax^2 + bx + c = 0 is given by:
S = −a/b

Summary

  • When b = a:
    S = -a/a = -1
  • When b = -a:
    S = a/a = 1

Thus, the sum of all solutions can be either -1 or 1
